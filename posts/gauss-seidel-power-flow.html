Gauss-Seidel Power Flow Method

2025-11-11 • Tutorial

The Gauss-Seidel method solves:

$$ V_i^{(k+1)} = \frac{1}{Y_{ii}} \left( \frac{P_i - jQ_i}{V_i^{(k)*}} - \sum_{j \neq i} Y_{ij} V_j^{(k)} \right) $$

MATLAB Code

Ybus = [...];
V = ones(n,1); theta = zeros(n,1);
for k = 1:100
  for i = 2:n
    sumYV = 0;
    for j = 1:n
      if j ~= i
        sumYV = sumYV + Ybus(i,j) * V(j) * exp(1j*theta(j));
      end
    end
    V(i) = (1/Ybus(i,i)) * ((P(i)-1j*Q(i))/conj(V(i)) - sumYV);
  end
end
